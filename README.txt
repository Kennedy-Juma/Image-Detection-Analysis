NOTE: Install Python

----Install needed libraries----
1. pip install opencv-python
2. pip install pytesseract
3.pip install webcolors
4. Download and configure Tesseract OCR as described in this stackoverflow comment: https://stackoverflow.com/questions/50951955/pytesseract-tesseractnotfound-error-tesseract-is-not-installed-or-its-not-i

----Running the application------
To run the application, type python manage.py runserver in command prompt/terminal.
